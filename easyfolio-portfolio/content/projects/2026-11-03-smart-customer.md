## 项目概述

基于大语言模型的智能客服系统，支持多渠道接入和智能工单分配。

## 项目背景

传统客服系统效率低下，无法满足企业日益增长的客户服务需求。本系统利用AI技术提升客服效率和客户满意度。

## 核心功能

1. 智能对话机器人
2. 多渠道统一管理
3. 智能工单分配
4. 客户满意度分析

## 设计亮点

- **智能路由**: 根据问题类型智能分配客服
- **知识库管理**: 支持快速知识查询
- **情绪分析**: 实时分析客户情绪
- **智能总结**: 自动生成对话总结

## 技术栈

- 前端: React + Redux
- 后端: Go + gRPC
- AI服务: 豆包大模型
- 数据库: Redis + MySQL

## 我的角色

作为产品设计师，负责产品策略、用户体验和界面设计。
