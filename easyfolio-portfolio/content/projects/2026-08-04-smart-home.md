## 项目概述

智能家居控制中心应用，统一管理各类智能设备。

## 项目背景

智能家居设备种类繁多，缺乏统一的管理入口。本应用旨在提供一个统一的智能家居控制平台。

## 核心功能

1. 设备管理与控制
2. 场景模式设置
3. 自动化规则配置
4. 能源消耗监控

## 设计亮点

- **统一控制**: 支持多种品牌设备
- **智能场景**: 一键切换场景模式
- **语音控制**: 支持语音指令
- **远程控制**: 随时随地控制设备

## 技术栈

- 前端: React Native
- 后端: Node.js + MQTT
- 云服务: AWS IoT
- 数据库: DynamoDB

负责界面设计和交互设计。
